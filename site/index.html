特徴

ファイル操作

コピー・移動・削除・リネーム・ディレクトリ作成・Zip 圧縮/解凍。長時間処理は進捗表示付きの非同期ジョブで実行し、Esc で中断できます。

シェル / コマンド実行

現在のディレクトリでシェルを起動（h）、任意のコマンドを実行（x）。ファイル操作とコマンドラインを行き来できます。

ファイル情報・属性

サイズ・種別・パーミッション・タイムスタンプなどの詳細（i）と属性パネル（a）をサイドパネルで確認できます。

Grep 検索

ディレクトリ配下を grep（g）。ヒットしたファイルへサイドパネルから直接ジャンプできます。

ブックマーク

よく使うディレクトリを登録（+）・削除（-）し、ブックマークパネル（b）から素早く移動できます。

プレビュー

テキスト・画像のプレビュー（v）に対応。オーディオファイルは再生・シークも可能です。

ツリー表示

ディレクトリ構造をツリー（t）で俯瞰し、展開・折りたたみしながら目的の場所へ移動できます。

検索・ジャンプ・Yank

インクリメンタル検索（f）、ディレクトリジャンプ（j）、選択パスのクリップボードコピー（y）。

インストール

Homebrew

macOS (Apple Silicon) / Linux (x86_64)

brew tap pkshimizu/tap
brew install fv

GitHub Releases

お使いのプラットフォームのアーカイブをダウンロードし、展開して fv を PATH の通った場所に置きます。

# macOS (Apple Silicon)
tar xzf fv-aarch64-apple-darwin.tar.gz
# Linux (x86_64)
tar xzf fv-x86_64-unknown-linux-gnu.tar.gz

mv fv /usr/local/bin/

主要なキー操作

ファイル一覧（メイン画面）での主なキー操作です。

移動

↑↓カーソル移動
←→先頭 / 末尾へ

Backspace親ディレクトリへ
<>履歴を戻る / 進む
~ホームへ
